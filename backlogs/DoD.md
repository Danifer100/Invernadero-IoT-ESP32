# Definition of Done (DoD) – Invernadero IoT

Para considerar que un incremento o historia está completada:  
1. El código fuente está revisado y comentado.  
2. El ESP32 se conecta de forma estable a WiFi.  
3. El sistema publica todos los datos por MQTT: temperatura, humedad relativa y humedad del suelo.  
4. El relé de riego responde correctamente a las condiciones de humedad establecidas.  
5. La simulación Wokwi está disponible (diagram.json) y ejecuta sin errores.  
6. Se ha probado funcionalidad de riego automático.  
7. Se ha creado el dashboard de Node-RED con gráficos y control de riego.  
8. Todos los entregables están documentados en el repositorio de GitHub.
